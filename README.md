## 📋 Sobre o projeto

Redesign do site oficial da **CIMATEC jr**, empresa júnior do SENAI CIMATEC. O objetivo é modernizar a identidade visual, melhorar a experiência do usuário e comunicar melhor os serviços e cases da empresa para potenciais clientes.

Projeto desenvolvido como atividade final do programa **Trainee CIMATEC jr 2026.1**.

---

## ✨ Funcionalidades base

- Design responsivo
- Acessibilidade básica
- Otimização de SEO
- Validação de formulário

## 🚀Roadmap

- Implementação de scroll suave
- Navbar fixa
- Animações CSS
